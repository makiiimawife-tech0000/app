# ⚡ Polymarket Arbitrage Bot

**Automated arbitrage trading for Polymarket prediction markets.**

---

## 🚀 Installation

**Double-click:**
```
INSTALL_AND_SETUP.bat
```

The installer:
- ✅ Checks for Python
- ✅ Installs all packages
- ✅ Creates desktop shortcut
- ✅ Launches the app

**That's it!**
